# Moonlight - iOS Monitor Support 🌙

A fork of [Moonlight iOS](https://github.com/moonlight-stream/moonlight-ios) 
with extra features for power users.

## ✨ New Features

- 🖥️ **External monitor fullscreen output** — dedicated output to monitor with waiting screen
- 🖱️ **Mouse cursor hidden** when streaming to external display
- 📱 **Touchpad Mode** — iPhone screen blacks out, use it as a trackpad
- 🔄 **Reverse scroll direction** toggle in settings
- 🎯 **Disable mouse smoothing** toggle in settings
- ⚡ **Lower latency optimizations** for smoother streaming
- 🌐 **Tailscale optimized** — better packet size for remote streaming

## 📲 How to Install

1. Download the latest `MoonlightPlus.ipa` from [Releases](../../releases/latest)
2. Install [AltStore](https://altstore.io) on your iPhone
3. Sideload the `.ipa` via AltStore
4. Re-sign every 7 days with AltStore (free Apple ID)

## ⚠️ Requirements

- iPhone with iOS 14+
- AltStore for sideloading
- A PC running [Sunshine](https://github.com/LizardByte/Sunshine) 
  or GeForce Experience

## 🔧 How to Build

1. Fork this repo
2. GitHub Actions automatically builds on every push
3. Download `.ipa` from Actions artifacts or Releases
